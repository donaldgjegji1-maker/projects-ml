A comprehensive data analysis project exploring the factors that contribute to gladiator success and survival in ancient Roman arenas. This project analyzes 9,976 gladiator records to uncover patterns in combat effectiveness, survival rates, and performance metrics.

##  Project Overview

This analysis examines a rich dataset containing 29 features per gladiator, including physical attributes, combat history, psychological factors, equipment quality, and survival outcomes. Through feature engineering and data visualization, we uncover what separates champions from casualties in the brutal world of gladiatorial combat.

##  Key Questions Answered

- **What weapons yield the highest victory counts?**
- **Which origins produce the most successful fighters?**
- **How does experience affect survival rates?**
- **What are the prime fighting years for gladiators?**
- **Which battle strategies are most effective?**

##  Technical Implementation

### Data Features
- **Demographic**: Age, Height, Weight, BMI, Origin
- **Combat**: Wins, Losses, Battle Experience, Weapon Choice
- **Psychological**: Mental Resilience, Public Favor
- **Equipment**: Weapon Quality, Armor Quality
- **Outcome**: Survival Status

### Feature Engineering
- Created meaningful categories (Age Groups, Experience Levels)
- Developed composite scores 
- Standardized weapon types and origin regions
- Implemented win ratios and fighting style classifications

### Visualization Highlights
- **6 comprehensive plots** telling the complete gladiator story
- Consistent color schemes with meaningful palettes
- Professional layout with clear value labels
- Multi-dimensional performance comparisons
